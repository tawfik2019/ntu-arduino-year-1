Python vs Arduino

The first conceptual demo of code for each section is done in Python (unless specified).

General comparison of techniques

For Arduino examples/explanations click 'Arduino Explanations' from the top right of the nav bar or click here.

Simple mode is just the comparison of code snippets. Very little extra text.

Layout

Python

myNumber = 5
if myNumber > 10:
    print("Greater than 10")
else:
    print("Less than 10")

Arduino

int myNumber = 5;
if (myNumber < 10) {
    Serial.print("Greater than 10");
} else {
    Serial.print("Less than 10");
}

Variables

Python

myInt = 5
myFloat = 5.0
myString = "Hello"

Generally:

<var_name> = <value>

Arduino

int myInt = 5;
float myFloat = 5.0;
String myString = "Hello";
Click here for more information regarding data types.

Generally:

<data_type> <var_name> = <value>

Constants

const keyword

const int myInt = 5;
const float myFloat = 5.0;
const String myString = "Hello";

Generally:

const <data_type> <const_name> = <value>;

#define

#define NUMBER 5
                    
// Using the value:
Serial.println(NUMBER);

Notice there is no '=' between the name and value.

The precompiler will convert the previous code snippit to the following: 
// Using the value:
Serial.println(5);

Generally:

#define <name> <value>

Using in code:

name

If Statements

Concept: 
if <condition>:
    // do this
else if <another_condition>:
    // do this instead
else:
    // do this if others all false

Python

myNumber = 5
if myNumber > 10:
    print("Greater than 10")
else:
    print("Less than 10")

Arduino

int myNumber = 5;
if (myNumber < 10) {
    Serial.print("Greater than 10");
} else {
    Serial.print("Less than 10");
}

While loops

Concept: 
while <condition>:
    // do this
    // can break out of while loop using 'break' (for both Python and Arduino)

Python

counter = 0
while counter < 5:
    print(counter)
    counter += 1

Arduino

int counter = 0;
while (counter < 5) {
    Serial.print(counter);
    counter++;
}

For loops

Concept: 
for <temp_variable> in range(<start>, <end>, <step>):
    // do this

Python

for i in range(1, 11, 2):
    print(i)

Arduino

Syntax:

for (<datatype> <var> = <start>; <var> < <end>; <var>++)

Example:

for (int i = 1; i < 11; i += 2) {
    Serial.print(i);
}

For vs While loops (Comparison)

While loops

int counter = 0;
while (counter < 5) {
    Serial.print(counter);
    counter++;
}

For loops

for (int i = 0; i < 5; i++) {
    Serial.print(i);
}

Functions

Concept: 
def <function_name>(<any_arguments_comma_separated>) :
    // do this
    // maybe return <value> (not neccesssary - function may just 'do' something i.e print something)

Python

Example of function that has return:

def add(num1, num2):
    result = num1 + num2
    return result

Example of function that has return:

def sayName(name):
    print(name)

Arduino

Syntax (data_type is void when function doesn't return anything, example below):

<data_type> <funct_name>(<data_type> <name_within_funct>, [more]) {
    // do this
    // maybe return <value>;
}

Example of function that has return:

float add(float num1, float num2) {
    float result = num1 + num2;
    return result;
}

Example of function that no return:

void sayName(String name) {
    Serial.println(name);
}
